# Feature: dataset

Генерирует обучающий датасет из эталонного CSV.

**Вход:** CSV, где колонки — это NER-метки (например, DRUG, DOSE, FORM).  
**Выход:** JSONL — каждая строка `{"text": "...", "entities": [[start, end, "LABEL"]]}`.

Что делает:
- Читает эталонный CSV и генерирует N вариаций на каждую строку
- Добавляет шум: опечатки, сокращения слов, удаление пробелов, перемешивание порядка
- Собирает готовые текст + спаны прямо здесь — training-фича получает финальный JSONL

Эндпоинт: `POST /generate/`
