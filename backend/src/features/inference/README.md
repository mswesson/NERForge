# Feature: inference

Разбирает текст с помощью обученной NER-модели.

**Вход:** zip-архив модели + произвольный текст.  
**Выход:** найденные сущности со span-координатами и список меток модели.

Что делает:
- Загружает spaCy-модель из zip (кэш по SHA-256 — повторная загрузка не перечитывает диск)
- При смене модели очищает предыдущую с диска
- Запускает NER в отдельном потоке (блокирующая CPU-операция)

Эндпоинт: `POST /parse/`
